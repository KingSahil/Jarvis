# WhatsApp Desktop Guide

This guide outlines common navigation methods and keyboard shortcuts for the WhatsApp desktop application (`whatsapp.root.exe`).

## ⚙️ Standard Behaviors

*   **Core Function:** WhatsApp is a messaging platform designed for real-time communication, allowing users to send and receive messages, make calls, and share media with contacts.
*   **Interface:** The application typically features a sidebar listing chats and a main panel for message viewing and composition.
*   **Efficiency:** The app heavily utilizes keyboard shortcuts to allow users to manage chats and perform actions without constant mouse interaction.

## ⌨️ Key Navigation Shortcuts

WhatsApp provides numerous shortcuts to enhance productivity. These shortcuts are crucial for navigating and managing the inbox:

*   **New Chat:** `Ctrl + N` (Windows)
*   **Mark as Unread:** `Ctrl + Alt + Shift + U`
*   **Mute Chat:** `Ctrl + Alt + Shift + M`
*   **Archive Chat:** `Ctrl + Alt + Shift + E`
*   **General Navigation:** Keyboard shortcuts can be used to navigate chats, send messages, and manage the inbox efficiently.

## 🧭 Common Navigation Paths

While specific menu paths can vary, the application's core functionality is accessed via:

*   **Starting a New Conversation:** Use the dedicated 'New Chat' shortcut (`Ctrl + N`) or the corresponding button/menu option.
*   **Managing Chats:** Chats can be managed by marking them as unread, muting them, or archiving them using the specific shortcuts listed above.
*   **Settings/Support:** Accessing settings or help features usually involves clicking the profile icon or navigating through the main menu structure, though specific shortcuts for these areas are not universally documented in the provided snippets.
