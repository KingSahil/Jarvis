# ChatGPT Desktop App Guide

This guide outlines common navigation methods and keyboard shortcuts for the ChatGPT desktop application.

## 💡 Standard Behaviors

*   **Input/Submission:** Use `Enter` (Windows) or `Enter` (Mac) to submit your input to the chat.
*   **Line Break:** Use `Shift + Enter` (Windows) or `Shift + Enter` (Mac) to add a line break within your input.
*   **Quitting:** Use `Alt + F4` (Windows) to close the application.

## ⌨️ Key Navigation Shortcuts

*   **Chat Bar Focus:** `Command + L` (Mac) or equivalent shortcut to focus on the chat input bar.
*   **Search:** `Command + F` (Mac) or equivalent shortcut to focus on the search bar.
*   **Model Picker:** `Command + J` (Mac) or equivalent shortcut to open the model selection menu.
*   **General Navigation:** The application features a dedicated keyboard icon in the navigation bar that provides a pop-up list of all available shortcuts on screen.

## 🧭 General Tips

*   **Windows Navigation:** For general system navigation (e.g., taskbar, command prompt), standard Windows shortcuts apply.
*   **Temporary Chat:** The app supports a mode for temporary or private chats (Incognito Mode).
