# Controlling Spotify via Antigravity.exe

## Standard Behaviors
- The application responds to various keyboard shortcuts for playback control and navigation.
- Global hotkeys are available for controlling Spotify.

## Navigation Paths
### Keyboard Shortcuts
- **Playback Control:**
  * Play/Pause: Spacebar or Ctrl + P
  * Next Track: Ctrl + Right Arrow or *
  * Previous Track: Ctrl + Left Arrow or /
  * Volume Up: Ctrl + Up Arrow or +
  * Volume Down: Ctrl + Down Arrow or -
- **Menu and Settings:**
  * Open Search: Ctrl + K or Cmd + K
  * Open Help: Ctrl + / or Ctrl + ? (on Windows)
- **Other Actions:**
  * Like/Dislike Song: Alt + Shift + B
  * Log Out: Alt + ... (specific key not provided, check app menu)

## Tips
- Ensure that global hotkeys are enabled in the application settings if shortcuts are not working.
- For more detailed shortcuts, refer to Spotify's official documentation or community forums.
