## 🧭 Known Navigation Paths & Shortcuts

### ⌨️ Global Windows Shortcuts (Highly Recommended)

*   **Alt Key Navigation:** Use the `Alt` key combined with the underlined letter of a menu item (e.g., `Alt + F` for File) to quickly jump to a menu. This is the most reliable way to navigate menus without a mouse.
*   **Task Switching:** Use `Alt + Tab` to switch between open applications.
*   **Search:** Use `Win` key + `S` (or the Start button search) to quickly find the app or files.

### 📂 Common App Menu Paths

*   **File Menu:** Typically contains actions like 'New', 'Open', 'Save', and 'Exit'. Use `Alt + F` to access this menu.
*   **Help/Support:** Look for a 'Help' or question mark (?) icon. This usually provides access to documentation or support contacts.
*   **Settings/Options:** Look for a gear icon (⚙️) or a 'File' menu option. This is where configuration changes are usually made.

### 💡 Pro Tip: Mastering the Keyboard

To maximize efficiency, remember that Windows allows deep navigation using only the keyboard. Practice using the `Alt` key combined with menu letters to bypass mouse usage entirely.
